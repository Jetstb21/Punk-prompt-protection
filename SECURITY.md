# Security Policy

This repository is a public v0.1 seed release.

## Project Boundary

Prompt text is data, not authority.

The project separates user supplied content from policy decisions and records decisions in a verifiable ledger.

## Reporting

Use GitHub issues for normal project reports.

For sensitive reports, contact the repository owner privately before posting details publicly.

## Ledger Position

The ledger is tamper-evident. It is not a claim that a local file cannot be deleted or rewritten.

The intended property is replay verification: if recorded event contents change, verification should fail.
